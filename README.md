Port Scanner:

Uno scanner di porte scritto in Python con supporto multithreading, sviluppato a scopo educativo per imparare i fondamentali del networking e della sicurezza informatica.

Funzionalità:

- Scansione automatica delle porte 1-1024
- Scansione manuale di porte specifiche
- Identificazione dei servizi comuni (HTTP, SSH, FTP, ecc.)
- Multithreading per scansioni veloci
- Output ordinato con nome del servizio associato ed in caso di servizio non presente nel dizionario verra mostrata in output solo la porta

Requisiti:

- Python 3.x
- Nessuna libreria esterna richiesta

Utilizzo:

Segui le istruzioni a schermo:
1. Inserisci l'indirizzo IP da scansionare
2. Scegli se scansionare tutte le porte 1-1024 oppure porte specifiche
